# **將檔案 push 到 GitHub 上的指定資料夾內**
- [在本地端建立資料夾](#在本地端建立資料夾)

## **在本地端建立資料夾**
- **git branch <分支名稱>**
    - 在 master 上切新分支。
    - 建立分支。
- **git checkout <分支名稱>**
    - 切換分支。
- 在與遠端儲存庫連結的本地端儲存庫內建立一個資料夾
- 新增檔案 → git add → git commit
- **git push -u <遠端名稱(origin)> <分支名稱>**
- 將檔案放入新建立的資料夾內
- **git add <資料夾名稱/檔案名稱.副檔名>**
- **git commit**
- **git push <遠端名稱(origin)> <分支名稱>**
