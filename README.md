# **測試**

## **將檔案 push 到 GitHub 上**
- **echo "# 標題" >> README.md**
	- 建立一個 README.md 檔，內容有一個標題
- **git remote add 遠端名稱 ( origin ) 複製網址**
	- 與遠端儲存庫連結	
- **git add 檔案名稱.副檔名**
	- 將檔案加入暫存區
- **git commit -m "版本訊息"**
	- 提交到儲存庫
- **git push 遠端名稱 ( origin ) 分支名稱 ( master )**
	- 將本地端的檔案推到遠端儲存庫
	- 登入：Sign in with your browser
	- 授權：Authorize git-ecosystem
	- push 成功：Authentication Succeeded
